## サポート対象
- Chromeブラウザ
- 日本語設定


## 使い方
1. ブラウザ右上のアイコン <img src="./docs/images/extension_icon.png" width="16"> をクリックします。
2. chat helper を選択します。ピン止めしておくと便利です。
3. 設定画面を開き、必要な設定をオンにします。
4. 設定を反映するために、chat画面を更新します。
